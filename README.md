# budaya-kerja-islami-di-era-saat-ini
Website statis ini merupakan projek integrasi antara kompetensi keahlian Teknik Jaringan Komputer dan Telekomunikasi (TJKT) dengan penerapan nilai-nilai karakter Islami. Fokus utama projek ini adalah mendemonstrasikan bagaimana infrastruktur digital dan publikasi informasi dapat dikelola.
